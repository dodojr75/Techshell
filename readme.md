Description: Home made shell that can navigate the terminal, create files, transfer information, and manage permisions
Instructions: 
  cd - go to home directory
  cd arg - go to directory
  ls - lists directories
  >, <, | - must have args on both sides
  chmod - change permission
  pwd - print working directory
What doesn't work: Navigating to files with spaces in the names
